# Restructure Scattered Material

Use this module when the user provides scattered素材, notes, rough scenes, bullet points, or a vague idea and asks to make it into a complete short-form video.

## Restructure Principle

Convert material into:

> 大钩子吸引人，小篇章推进内容，小钩子维持兴趣，结尾落点留下记忆。

Do not preserve the original order by default. Choose the order that creates the clearest progression and viewer gain.

## Workflow

1. List the usable material: scenes, actions, objects, sounds, emotions, facts, opinions, or transitions.
2. Choose one core theme.
3. Choose one viewer gain.
4. Select a progression model:
   - 混乱 -> 有序 -> 平静 -> 安定感
   - 问题 -> 尝试 -> 变化 -> 解决
   - 普通日常 -> 细节发现 -> 情绪放大 -> 主题回扣
   - 期待 -> 落差 -> 调整 -> 收获
   - 观察 -> 解释 -> 例子 -> 方法
   - 状态 A -> 行动 -> 状态 B -> 感受
5. Build a big hook from the most watchable tension, result, image, emotion, or question.
6. Split the middle into 3-5 chapters. Each chapter must have a small task.
7. Put a small hook between chapters.
8. End with a payoff that names or implies the viewer gain.

## Chapter Rules

Each chapter must answer:

- 这一段在推进什么？
- 这一段为什么让人继续看？
- 这一段和主题有什么关系？

If a chapter cannot answer these three questions, merge it, cut it, or turn it into a concrete detail inside another chapter.

## Deliverable

When the user wants a practical result, provide:

- A concise structure table.
- A 30-60 second script version if duration is unknown.
- Optional alternative openings if the hook could go in several directions.
