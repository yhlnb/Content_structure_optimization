# Diagnose Existing Video Structure

Use this module when the user provides an existing script, outline, shot list, draft caption, or content idea and asks whether it works, where it feels scattered, or how to improve it.

## Diagnosis Steps

1. Extract the likely core theme in one sentence.
2. State the likely viewer gain.
3. Check whether the opening gives a reason to stay in the first 3-5 seconds.
4. Map the middle into chapters and identify whether each chapter advances event, emotion, viewpoint, or character state.
5. Check whether chapter transitions contain small hooks.
6. Check whether concrete details support the theme.
7. Check whether the emotional line changes naturally.
8. Check whether the ending returns to the theme or forms a clear feeling.
9. Give a revised structure direction before line-level wording suggestions.

## Diagnosis Output

Use this structure:

```markdown
## 一句话判断

## 主要问题
| 问题 | 为什么影响观看 | 修改方向 |
|---|---|---|

## 结构诊断
| 标准 | 当前判断 | 建议 |
|---|---|---|

## 建议重构方向

## 可直接修改的版本
```

## Common Problems

- Theme is too broad: narrow it to one sentence the viewer can repeat.
- Opening is descriptive but not motivating: add emotion, problem, result, image, suspense, or viewpoint.
- Middle is a timeline only: convert it into state change, problem solving, emotional progression, or viewpoint deepening.
- Chapters are decorative: require each chapter to answer what it advances, why viewers keep watching, and how it serves the theme.
- Details are abstract: add visible actions, objects, sounds, scenes, expressions, or perceptible changes.
- Emotion jumps too fast: insert a transition scene or sentence that explains the change.
- Ending stops instead of lands: return to the theme with a viewpoint, feeling, action, life observation, or memory point.

## Scoring

When useful, score each item from 1-5: theme clarity, opening hook, middle progression, chapter layering, small hooks, concrete details, emotional continuity, pacing, ending payoff, audiovisual consistency, and viewer gain. Keep scoring secondary to actionable rewrite advice.
