# Final Checklist

Use this module for final review, scoring, or acceptance checks.

## Core Standard

A good short-form video lets the viewer gain something clear in a short time. The gain may be knowledge, emotion, aesthetics, resonance, inspiration, companionship, or action desire.

## Self-Check Table

| 标准 | 自检问题 | Good Sign | Fix If Weak |
|---|---|---|---|
| 主题明确 | 这条视频能否用一句话概括？ | One clear sentence | Narrow to one subject and one viewer gain |
| 开头有钩子 | 前 3-5 秒是否有观看理由？ | Viewer knows why to stay | Add emotion, problem, image, suspense, result, or viewpoint |
| 中间有推进 | 内容是否从一个状态走向另一个状态？ | Visible change or deepening | Convert timeline into progression |
| 内容有层次 | 中间是否拆成几个小篇章？ | Each chapter has a task | Split into background, action, change, feeling, or return |
| 小钩子连接 | 每个篇章之间是否有继续看的理由？ | Transitions create expectation | Add question, unfinished action, contrast, or turn |
| 细节具体 | 是否有具体动作、物件、声音或画面？ | Viewer can picture it | Replace abstract wording with sensory detail |
| 情绪连贯 | 情绪是否自然流动？ | Feeling changes step by step | Add bridging action or observation |
| 节奏清楚 | 开头、中段、结尾是否节奏分明？ | Content keeps moving | Tighten opening, steady middle, slow ending |
| 结尾有落点 | 最后是否回到主题或形成感受？ | Viewer remembers the point | Add viewpoint, emotion, action, life feeling, or theme return |
| 视听统一 | 画面、音乐、字幕、口播是否服务同一氛围？ | One clear tone | Align wording, music, visual choices |
| 有收获感 | 观众看完后能获得什么？ | Gain is explicit or felt | State or imply the takeaway more clearly |

## Final Acceptance

Before finishing, ensure the response helps the user answer:

- 这条视频在讲什么？
- 观众为什么要继续看？
- 中间怎样一步步推进？
- 哪些细节让内容可信、可感、可记？
- 结尾为什么不是突然停止？
- 观众看完后留下什么？

If the answer is still vague, recommend another structural pass before polishing language.
