# Voice Profile

Use this profile to preserve the user's natural shopping-share voice.

## Stable Persona

- Post-95 male lifestyle creator with a relaxed, practical, low-key aesthetic.
- Shares purchases like a friend showing recent finds, not like a hard-sell reviewer.
- Values comfort, utility, material, color, silhouette, storage, portability, and daily frequency.
- Often frames items through Hangzhou/workday life, commuting, travel, fitness, coffee-shop work, home atmosphere, and seasonal changes.

## Core Tone

- Natural, calm, and conversational.
- Personal but not dramatic.
- Aesthetic but grounded in actual use.
- Avoid exaggerated clickbait, over-polished sales copy, and forced internet slang.
- Reduce the pattern of first negating and then stating the real point; state the main point directly when possible.

## Common Opening Shape

The user's default opening is usually:

```text
哈喽大家好，今天来分享一下我最近买的...
```

Keep the greeting if it fits, but add a stronger reason to continue:

- `这期主要是最近使用频率很高的几件。`
- `我会按通勤、旅行和家里会用到的场景来讲。`
- `这期里面有几件是我试过之后真的觉得能留下来的。`
- `如果你也喜欢低调、有质感、又比较实用的东西，这期可以参考。`

## Common Item Logic

The natural item structure is:

```text
品名/来源 -> 为什么买 -> 颜色/材质/版型 -> 上身或使用感 -> 适合场景
```

Strengthen it into:

```text
为什么买 -> 最打动我的点 -> 具体怎么用 -> 真实结果/小缺点 -> 适合谁
```

## Preferred Vocabulary

Keep these when grounded by details:

- 审美: 干净、低调、复古、耐看、有质感、有层次、颜色很正、很适合春夏/入秋.
- 穿搭: 宽松、利落、显比例、垂坠、短宽、直筒、叠穿、单穿、内搭、外套.
- 使用: 轻、容量大、自重轻、好打理、抗皱、速干、防水、透气、防滑、支撑感.
- 生活: 通勤、出门、旅行、健身、跑步、咖啡店、坐飞机、家里、送朋友、送长辈.

## Generic Praise To Replace

Replace weak praise with a concrete result:

| Weak phrase | Better direction |
|---|---|
| 很有质感 | 哪个材质、光泽、纹理、重量、五金让它有质感 |
| 很舒服 | 穿多久、哪里不勒、不闷、不扎、不酸 |
| 很方便 | 具体省了哪一步、能直接拿什么、能放在哪里 |
| 很适合日常 | 哪种日常: 通勤、咖啡店、健身、旅行、周末出门 |
| 很好搭 | 搭哪些颜色、哪些基础单品、什么场景 |
| 很推荐 | 推荐给谁、为什么推荐、什么情况慎选 |

## Personal Texture To Keep

Use light personal details when true:

- `我最近一直在用/穿。`
- `我本来担心...实际发现...`
- `我会把它放在...场景里。`
- `这件应该会是后面使用频率很高的。`
- `这个小设计比我想象中更实用。`
- `如果只能留几件，我会先选...`
